# emlix GmbH

**Hiring fit:** Yes - strong industrial Embedded Linux specialist with remote/Munich options and a clear security lifecycle niche.

## Company snapshot

emlix GmbH is a German embedded Linux specialist headquartered in Göttingen, with locations including Berlin, Bonn, and Munich plus home-office options. It develops industrial-grade embedded Linux systems for connected devices, machines, and plants.

The company works on Linux bootloaders, kernel and driver development, Yocto/Buildroot/PTXdist, Qt/QML, Rust/C/C++, OTA updates, hypervisors, Linux containers, security engineering, CVE management, CRA/RED compliance, IEC 62443-4-1 processes, IEC 62304-related environments, automated testing, and lifecycle maintenance.
